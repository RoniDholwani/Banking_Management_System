# 🏦 Banking Management System

A complete **Banking Management System** developed using **Java**, **JDBC**, and **Swing GUI** that allows users to securely manage bank accounts, perform transactions, and monitor balances through an interactive desktop interface.

---

## ✨ Features

* 🔐 Secure Login & Authentication
* 👤 Create New Bank Accounts
* 💰 Deposit Money
* 💸 Withdraw Money
* 🔄 Transfer Funds Between Accounts
* 📊 Check Account Balance
* 🗄️ MySQL Database Integration using JDBC
* 🖥️ Interactive Swing GUI Interface
* ⚡ Real-time Transaction Processing
* 🔒 Security PIN Verification
* 🧾 Transaction Validation & Error Handling

## 🚀 Getting Started

### 1️⃣ Clone Repository

```
git clone https://github.com/RoniDholwani/Banking_Management_System.git
```

### 2️⃣ Open Project

Open the project in:

* IntelliJ IDEA
* Eclipse
* NetBeans

---

### 3️⃣ Configure MySQL Database

Create a MySQL database and required tables.

Example:

```
CREATE DATABASE banking_system;
```

Update database credentials inside your Java files:

```
String url = "jdbc:mysql://localhost:3306/banking_system";
String user = "root";
String password = "your_password";
```

---

### 4️⃣ Run the Project

Run:

```
Main.java
```

or launch the GUI application from:

```
BankingApp.java
```
